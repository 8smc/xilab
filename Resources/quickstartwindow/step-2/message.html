Connecting a controller via Ethernet

If you are using an Ethernet controller:

  1. In the XILab start window, click "Settings >>".
  2. Enable the "Enumerate network devices" flag.
  3. In the "Protocol" field, select the communication protocol from the drop-down list.
  4. In the "IP/Host" field, enter the IP address of your controller. To detect the controller in your local network, you can use Revealer.
  5. In the XILab start window, click the "Rescan" button. Select one or more controllers from the list using the CTRL or Shift buttons.
  6. Click the "Open selected" button
