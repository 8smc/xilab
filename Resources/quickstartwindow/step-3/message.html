Loading a profile

  1. In the XILab main window, click the "Settings..." button.
  2. Click "Load setting from file..." button.
  3. In the STANDA folder, find and select the profile for your stage and "Open" it.
  4. Click "Apply". Optionally, use "Save to flash" to save settings to the controller's flash.

Note: If you have opened XILab in multi-axis mode, you need to load the profile for each axis.
